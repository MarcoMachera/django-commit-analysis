┌─────────────────────────────────────────────────────────────┐
│                  FLUSSO DI LAVORO                           │
└─────────────────────────────────────────────────────────────┘

1️⃣  ESTRAZIONE DATI
    git log --pretty=format:"%H;%an;%ad;%s" > commits.csv

2️⃣  INSTALLAZIONE DIPENDENZE
    pip install -r requirements.txt

3️⃣  ANALISI BASE
    python3 analisi_commit.py
    → Genera: grafico_commit.png, commit_refactoring.csv

4️⃣  ANALISI AVANZATE
    python3 analisi_avanzate.py
    → Genera: 3 grafici aggiuntivi + statistiche console

5️⃣  PRESENTAZIONE
    Apri report_analisi.html nel browser
    Consulta RIEPILOGO.md per i dati chiave
